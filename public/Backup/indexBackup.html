About Me

My name is Matthew Sullivan...

I'm a web developer and video game programmer. I consider myself to be equal parts logical and creative. I'm proficient with modern languages such as JavaScript as well as foundation languages like C. I enjoy the challenge of integrating new technologies in a way that complements what I've already learned and thus, I keep learning.


Although I'm comfortable with general web development, my real passion is coding video games. My first major project, Firewall Penitentiary, is a real-time, multiplayer on-line e-sports style game. It was coded in Javascript using Node.js with socket.io used to communicate between server and client.

To see what project(s) I'm working on these days, feel free to check my blog page.
